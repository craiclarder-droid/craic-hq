# Deploy Craic HQ to GitHub Pages

1. Open your `craic-hq` repository.
2. Choose **Add file → Upload files**.
3. Upload all files from this folder to the repository root.
4. Commit the changes.
5. Open **Settings → Pages**.
6. Set Source to **Deploy from a branch**.
7. Select `main` and `/(root)`, then Save.
8. Wait a few minutes for the published URL.
9. Open the URL in Safari on iPhone.
10. Tap Share → Add to Home Screen.

## Data
This build stores data locally in the browser on each device.
Use Backup → Export full backup regularly.
