# Craic HQ 1.1 — Connected Production

This update preserves the existing `craic_hq_v2` browser database and adds:

- Persistent multi-blend production plans
- Combined ingredient and packaging totals
- Shopping statuses: Needs ordering, In basket, Ordered, Part received, Received
- Production queue that can be completed over several days
- Individual completed batches and automatic finished batch codes
- Actual yield and wastage
- Exact supplier-lot deduction
- Automatic stock updates
- Production-plan links in traceability
- HACCP records linked automatically by production date
- Recipe snapshots preserved for old batches
- Version display and safe backup imports

Before updating, export a full backup. Upload all files to the GitHub repository root and replace the existing files.
